# Fachinformatiker Anwendungsentwicklung Wiki

[Click here to visit the live site](https://discohagen.github.io/FIAE/)

## Prerequisites

The project uses:

- Python 3.12.X
- Git 2.43.X

## Getting Started

1. Clone the Repositoty:

```bash
git clone git@github.com:discohagen/FIAE.git
cd FIAE
```

2. Create and Activate a Python Virtual Environment

```bash
python -m venv .venv
source .venv/bin/activate
```

3. Install Dependencies

```bash
pip install -r requirements.txt
```

## Local Development

To serve locally:

```bash
mkdocs serve
```

To build:

```bash
mkdocs build
```

To deploy:

```bash
mkdocs gh-deploy
```
